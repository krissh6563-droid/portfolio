Formal Verification of Red-Black Tree implementation

We analyzed the function _Rb_tree_insert_and_rebalance in the STL library implementation which inserts a new node into the red-black tree and rebalances if necessary. We verified that after insertion of a new node, the new tree still satisfies some constraints of a red-black tree. We used a separation logic based verifier Verifast to write preconditions and postconditions for validation.

Performance monitoring tool using PMU

We interfaced with the PMU provided by modern CPUs to sample the occurrences of various events in the system. A linux kernel module was written which calculates the amount of time spent in each instruction function of a program; and the number of L1 caches misses, branch mispredictions at instruction level. A comparison with the the tool perf was done to validate the results.

Global variable accesses made in procedures

An access based interprocedural analysis was done to partition the data space for procedures based on the variables accessed within a procedure and its callees. We discovered access of variables locally and propagated information over the call graph. Implemented this module as a plugin in gcc-4.7.2 by analyzing GIMPLE, the intermediate representation (IR) of GCC.

Modeled and Animated a Transformer using OpenGL

We created an interactive transformer model using OpenGL primitives. A basic animation recording interface to capture keyframes was implemented. We created a short animation movie by interpolating the captured key-frames. Learned about Modeling Viewing Pipeline, 3D Transformations, Shading and Animation.

Song Genre Classification based on Audio Spectrum

We classified the songs according to the genre by analyzing their audio spectrum. The tool YAAFE was used for extracting various features of an audio such as ZCR, loudness, spectral flux, spectral roll-off, timbre. We implemented and experimented with various classification algorithms such as SVM, Neural Network, Random Forest in R. An accuracy of 85% was achieved on the GTZAN standard song dataset.

Network simulation trace analysis with NS3

Automated the process of analyzing a network simulation trace and generating summary reports by making use of sed, awk, python, pyplot and bash.

REPA library in Haskell

REPA is a Haskell library for high performance, regular, multi-dimensional parallel arrays. We implemented a subset of the functionalities provided by REPA library. We analyzed the effect of parallelization while running various types of algorithms that involve arrays.
